# Fitness Goal Tracking Web App

A modern and responsive gym-focused web application developed as part of a web programming course. This project is designed to help users manage their fitness journey by setting goals, tracking achievements, saving favorites, and viewing progress statistics through an interactive multi-page platform.

## Overview

This project was created to apply core front-end development principles while building a practical fitness management application. It focuses on responsive design, user-friendly navigation, and interactive features that support users in monitoring their workout progress and staying motivated.

## Key Features

* Responsive and intuitive user interface
* Multi-page website structure
* Fitness goal setting and tracking
* Achievements and milestone progress
* Favorites section for preferred workouts or routines
* Statistics dashboard for tracking progress
* Smooth navigation and interactive elements

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

* `index.html` – landing page
* `Home.html` – main dashboard
* `Goals.html` – fitness goals management
* `achivements.html` – achievements and milestones
* `favorites.html` – saved workouts or favorites
* `statistics.html` – fitness progress and insights
* `css/` – styling files
* `js/` – JavaScript functionality
* `images/` – project assets

## Getting Started

1. Clone or download the repository.
2. Open the project folder.
3. Launch `index.html` in your preferred browser.
