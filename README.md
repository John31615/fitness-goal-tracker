# 🏋️ Fitness Goal Tracking Web App

A modern, responsive fitness web application designed to help users manage their fitness journey by setting goals, tracking achievements, saving favorites, and monitoring progress through an interactive multi-page platform.

---

## 📌 Overview

This project was developed as part of a web programming course to apply core **front-end development principles** in a practical, real-world fitness tracking application.

It focuses on:

* Responsive design
* Clean and intuitive navigation
* Interactive user experience
* Structured multi-page architecture

The goal of the application is to help users stay motivated and consistently track their fitness progress.

---

## ✨ Key Features

* 📱 Fully responsive and user-friendly interface
* 📄 Multi-page website structure for better organization
* 🎯 Fitness goal setting and tracking system
* 🏆 Achievement and milestone tracking
* ⭐ Favorites section for workouts and routines
* 📊 Progress and statistics dashboard
* 🔄 Smooth navigation and interactive UI elements

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

```
index.html          → Landing page  
Home.html           → Main dashboard  
Goals.html          → Fitness goal management  
achievements.html   → Milestones and achievements  
favorites.html      → Saved workouts and routines  
statistics.html     → Progress tracking and insights  
css/                → Styling files  
js/                 → JavaScript functionality  
images/            → Project assets  
```
